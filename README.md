# PhotoGetter

photoGetter是一款自动爬取网页中图片的工具。它基于urllib和正则表达式实现网页图片的快速获取。

# 入门

photoGetter主要包含Spider和Main。

Spider主要是实现网页信息的爬取。

Main主要是实现爬取信息的整理和总体程序的管理。

# 文件结构

main.py包含程序的源代码。

# 支持平台

photoGetter基于Python3.5。如果想要运行photoGetter推荐下载Python3.x解析器，并且需要urllib包的支持。同时，需要注意处理文件和网页的格式。
